# 📦 iGrafic360 - Ecosistema de Logística Adaptado (VZLA-Ready)

Solución integral Fullstack para la gestión de tracking internacional, diseñada específicamente para solventar los retos operativos y financieros del mercado venezolano.

## 🇻🇪 Especialización para Venezuela
- **Gestión Multimoneda:** Soporte nativo para pagos en **Bolívares (Bs.)** y **Dólares (USD)**.
- **Tasa Automatizada:** Integración de calculadora con tasa **BCV / Euro** actualizada en tiempo real para clientes.
- **Pasarela de Pagos:** Soporte para **Pago Móvil** y **Binance Pay**, facilitando la conciliación administrativa.
- **Logística Puerta a Puerta:** Optimizado para el flujo Miami -> Aduana -> Entrega Nacional.

## 🚀 Arquitectura Tecnológica
- **App Mobile (Flutter):** Aplicación nativa de alto rendimiento para el cliente final.
- **Web Dashboard (React):** Panel administrativo VIP con estadísticas de ingresos y control de inventario.
- **Backend Robusto (Node.js & Python):** API principal y microservicios de automatización de datos.
- **Base de Datos (MySQL):** Estructura relacional para un historial de tracking infalible.

---

## ✨ Funcionalidades Estrella
- **QR Tracking:** Generación y escaneo de códigos QR para ingreso rápido a bodega.
- **Evidencia en Galería:** Visualización de fotos reales del paquete con sistema de carga optimizada.
- **Panel Administrativo:** Control total de paquetes "Sin Identificar" y asignación rápida a clientes.
